# 文章阅读体验修正 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修正文章详情页中段返回不便和继续阅读提示不明显的问题。

**Architecture:** 仅修改前端文章详情页。返回入口增加悬浮按钮并复用原 `goBack` 逻辑；继续阅读提示由内联提示条改为 Element Plus 弹窗，复用现有 `showResumePrompt`、`continueReadingFromSavedPosition` 和 `dismissResumePrompt` 状态。

**Tech Stack:** Vue 3、Vue Router、Element Plus、Node.js test runner、Vite

---

## Task 2: 实现文章详情交互

- [ ] **Step 1: 增加弹窗和悬浮返回模板**

在 `ArticleDetail.vue` 中移除 `.resume-reading-panel` 内联区域，改为文章模板后方的 `el-dialog`。在现有返回顶部按钮旁增加悬浮返回按钮。

- [ ] **Step 2: 增加状态与方法**

新增 `showFloatingBackButton`，用 `showBackToTop` 的滚动阈值同步；新增 `showResumeDialog` computed；新增 `handleFloatingBack`，先 `flushReadingPosition()` 再 `goBack()`。

- [ ] **Step 3: 调整样式**

删除 `.resume-reading-panel` 样式，新增 `.resume-reading-dialog`、`.resume-dialog-copy`、`.floating-reading-tools`、`.floating-back-button` 样式。移动端保持按钮不遮挡正文，支持 reduced motion。
