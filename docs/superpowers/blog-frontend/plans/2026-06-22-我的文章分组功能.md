# 我的文章分组功能 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为登录用户增加“我的文章分组”能力，支持创建、重命名、删除分组，并在写文章和我的文章列表中使用分组管理个人文章。

**Architecture:** 后端新增个人分组表和文章分组关联表，关联表预留一篇文章多个分组的能力；本期前端只提供单选分组体验，未选择分组的文章归入虚拟“未分组”。公开文章列表和公开标签逻辑不受影响。

**Tech Stack:** Spring Boot、MyBatis-Plus、MySQL、JUnit、Vue 3、Element Plus、Axios。
