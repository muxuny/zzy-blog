# Article Detail Continuation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为文章详情页增加上一篇、下一篇、相关文章、悬浮回顶和目录当前章节高亮。

**Architecture:** 后端在公开文章接口下新增相邻文章和相关文章查询能力，复用已发布文章可见性规则，不新增数据库表。前端详情页在当前文章加载成功后并行加载推荐数据，并用滚动监听维护目录高亮和悬浮回顶按钮状态。

**Tech Stack:** Spring Boot 2.7、MyBatis-Plus、JUnit 5、Mockito、Vue 3、Vue Router、Element Plus、Vite、Node test。

新增相关文章查询逻辑：

```java
@Override
public List<Article> getPublicRelatedArticles(Long id, Integer size) {
    Article current = requirePublicArticle(id);
    int limit = size == null || size <= 0 ? 4 : Math.min(size, 12);
    List<Long> tagIds = getTagIds(current.getId());
    if (tagIds.isEmpty()) {
        return java.util.Collections.emptyList();
    }
    List<Long> candidateIds = articleTagMapper.selectList(
            new LambdaQueryWrapper<ArticleTag>().in(ArticleTag::getTagId, tagIds))
            .stream()
            .map(ArticleTag::getArticleId)
            .filter(articleId -> !articleId.equals(current.getId()))
            .distinct()
            .collect(Collectors.toList());
    if (candidateIds.isEmpty()) {
        return java.util.Collections.emptyList();
    }
    List<Article> candidates = list(new LambdaQueryWrapper<Article>()
            .in(Article::getId, candidateIds)
            .eq(Article::getStatus, ArticleStatus.PUBLISHED));
    java.util.Map<Long, Long> sharedCounts = articleTagMapper.selectList(
            new LambdaQueryWrapper<ArticleTag>()
                    .in(ArticleTag::getArticleId, candidateIds)
                    .in(ArticleTag::getTagId, tagIds))
            .stream()
            .collect(Collectors.groupingBy(ArticleTag::getArticleId, Collectors.counting()));
    return candidates.stream()
            .sorted(java.util.Comparator
                    .comparing((Article article) -> sharedCounts.getOrDefault(article.getId(), 0L)).reversed()
                    .thenComparing(Article::getCreatedAt, java.util.Comparator.nullsLast(java.util.Comparator.reverseOrder()))
                    .thenComparing(Article::getId, java.util.Comparator.reverseOrder()))
            .limit(limit)
            .peek(this::attachTagsAndAuthor)
            .collect(Collectors.toList());
}
```

新增 helper：

```java
private Article findPreviousPublicArticle(Article current) {
    List<Article> articles = list(new LambdaQueryWrapper<Article>()
            .eq(Article::getStatus, ArticleStatus.PUBLISHED)
            .and(wrapper -> wrapper
                    .lt(Article::getCreatedAt, current.getCreatedAt())
                    .or()
                    .eq(Article::getCreatedAt, current.getCreatedAt())
                    .lt(Article::getId, current.getId()))
            .orderByDesc(Article::getCreatedAt)
            .orderByDesc(Article::getId)
            .last("LIMIT 1"));
    return articles.isEmpty() ? null : articles.get(0);
}

private Article findNextPublicArticle(Article current) {
    List<Article> articles = list(new LambdaQueryWrapper<Article>()
            .eq(Article::getStatus, ArticleStatus.PUBLISHED)
            .and(wrapper -> wrapper
                    .gt(Article::getCreatedAt, current.getCreatedAt())
                    .or()
                    .eq(Article::getCreatedAt, current.getCreatedAt())
                    .gt(Article::getId, current.getId()))
            .orderByAsc(Article::getCreatedAt)
            .orderByAsc(Article::getId)
            .last("LIMIT 1"));
    return articles.isEmpty() ? null : articles.get(0);
}

private List<Long> getTagIds(Long articleId) {
    return articleTagMapper.selectList(
            new LambdaQueryWrapper<ArticleTag>().eq(ArticleTag::getArticleId, articleId))
            .stream()
            .map(ArticleTag::getTagId)
            .collect(Collectors.toList());
}
```

```js
export function getActiveHeadingId(headings = [], scrollY = 0, offset = 120) {
  if (!Array.isArray(headings) || headings.length === 0) return ''

  const currentY = scrollY + offset
  let active = headings[0]?.id || ''

  for (const heading of headings) {
    if (typeof heading?.top !== 'number') continue
    if (heading.top <= currentY) active = heading.id
    else break
  }

  return active
}
```

async function loadArticle() {
  loading.value = true
  errorMessage.value = ''
  try {
    const r = await getArticle(route.params.id)
    article.value = r.data
    await nextTick()
    updateScrollState()
    loadContinuationData(r.data.id)
  } catch {
    article.value = null
    errorMessage.value = '文章不存在或暂不可见'
  } finally {
    loading.value = false
  }
}

async function loadContinuationData(articleId) {
  try {
    const [neighborsResult, relatedResult] = await Promise.all([
      getArticleNeighbors(articleId),
      getRelatedArticles(articleId, 4)
    ])
    neighbors.value = neighborsResult.data || { previous: null, next: null }
    relatedArticles.value = relatedResult.data || []
  } catch {
    neighbors.value = { previous: null, next: null }
    relatedArticles.value = []
  }
}

function updateScrollState() {
  showBackToTop.value = window.scrollY > 360
  if (!toc.value.length) {
    activeHeadingId.value = ''
    return
  }
  const headings = toc.value
    .map(item => {
      const el = document.getElementById(item.id)
      return el ? { id: item.id, top: el.getBoundingClientRect().top + window.scrollY } : null
    })
    .filter(Boolean)
  activeHeadingId.value = getActiveHeadingId(headings, window.scrollY, 140)
}

function openArticle(id) {

function scrollToTop(smooth = true) {
